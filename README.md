# KSS PC Club - Website

## このリポジトリは？

古河中等パソコン部の Web サイトです。

https://kss-pc.club/

## 開発方法

1. Node.js と Yarn をインストールします。
2. このリポジトリを Clone します。
3. `yarn install --frozen-lockfile` します。
4. `yarn husky:prepare` でコミット前の自動整形等の準備をします。
5. `yarn start` でローカルサーバー（http://localhost:8000/）を立ち上げます。
